一些vbs小资源


vbs是什么？

天影大侠帮你摘抄了百度百科里关于vbs的第一句话

VBS是基于Visual Basic的脚本语言。VBS的全称是：Microsoft Visual Basic Script Edition。（微软公司可视化BASIC脚本版）。

所以，从这句话中可以得出，vbs是脚本语言

大家可以百度查查关于vbs的其他信息，天影大侠这里贴心地准备了百度百科链接...

当然了，和你们一样，我只是一个新手，碰vbs都是为了整蛊好友们

所以，这里就分享天影大侠在网上找到的一些vbs资源

可以直接网盘下载，也可以复制代码，放在一个TXT文本里，保存后将后缀名改为.vbs即可！！！

1.一个问题，看图，长这样，如不选择"是"，则永不能退出！

链接：https://share.weiyun.com/mE33CTeA 密码：wzh666

懒得下载？

好的，这里有代码：（中文的地方可以自行修改）

do
a =msgbox("天影大侠厉害不？？？",vbYesNoCancel,"一个问题")
if a =vbYes then
    msgbox "天影大侠厉害！！！",,"感谢你的赞扬！！！"
exit do
elseif a =vbNo then
    b =msgbox("你完了",vbYesNo,"你完了。")
    if b =vbYes then
    msgbox "你完了！",,"你完了！！！"
    else
    msgbox "你完了！！！",,"你完了。。。"
    end if
else
    d =msgbox("你完了！！！",vbYesNo,"你完了——")
    if d =vbYes then
    msgbox "你完了！",,"你完了！！！"
    else
    msgbox "你完了。。。",,"你完了！！！"
    end if
end if
loop
